# PUBLOX TENERIFE

Projet Next.js 14 avec TypeScript, TailwindCSS, i18n (next-intl), et animations Framer Motion.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) - vous serez automatiquement redirigé vers `/es`.

## Build

```bash
npm run build
npm start
```

Voir `SPEC.md` pour la documentation complète.

- Next.js 14 (App Router)

- TailwindCSS

- Framer Motion
